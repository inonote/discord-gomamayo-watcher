# discord-gomamayo-watcher
ゴママヨ絵文字リアクションを検出する Discord Bot。ゴママヨ以外にも使えます。

## 使い方
1. .env 色々書く
2. bot 起動
3. サーバーに招待
4. スラッシュ コマンドで色々設定する
5. 完了

## インストール
```
npm i
```

## 実行
```
npm run build
npm run start
```

## 開発
ホット リロードが効きます。

```
npm run dev
```

## フォーマット
```
npm run format
```
