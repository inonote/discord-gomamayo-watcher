# discord-gomamayo-watcher
ゴママヨ絵文字リアクションを検出するくん。ゴママヨ以外にも使えます。

## 使い方
1. サーバーに招待
2. スラッシュ コマンドで色々設定する
3. 完了

## インストール
```
npm i
```

## 実行
```
npm run build
npm run start
```

## 開発
ホット リロードが効きます。

```
npm run dev
```

## フォーマット
```
npm run format
```
